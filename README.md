This project is a lightweight train ticketing system implementing user authentication, schedule CRUD operations, and seat inventory management without external frameworks. built with native PHP and MySQL. 
It facilitates a complete booking workflow where users can search for train schedules, select specific classes (first, second, third), and book available seats via an interactive AJAX-based interface.

The key Features:

Role based authentication: Login system supporting administrators, workers, and passengers.
Admin dashboard: Capabilities to manage trains, routes, schedules, and view all user bookings.
Worker portal: Dedicated view for staff to inspect issued tickets and passenger details.
User dashboard: Allows passengers to view available rides and track their booking history.
Dynamic booking: Real-time checking of seat availability and PNR generation upon confirmation.
